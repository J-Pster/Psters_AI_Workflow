# Execute Work

**Objective:** Execute free-form implementation work with research, structured tasks, and validation.

**Inputs:** What to implement or fix.

**Steps:**
1. Clarify objective and expected outcome.
2. Research code context and constraints first.
3. Derive concrete task list with file targets.
4. Implement task-by-task and validate after changes.
5. Summarize modified files, outcomes, and follow-ups.

**Output:** Summary of modified files, outcomes, and follow-ups.
