# Compound Knowledge

**Objective:** Capture solved problems or reusable patterns as durable team knowledge.

**Inputs:** Optional context or `pattern` mode.

**Steps:**
1. Determine if this is a solved problem or a reusable pattern.
2. Collect root cause, implementation approach, and key constraints.
3. Synthesize one high-signal knowledge artifact.
4. Classify and store it under the correct documentation area.
5. Link related docs to improve future discovery.

**Output:** Knowledge artifact stored and linked.
