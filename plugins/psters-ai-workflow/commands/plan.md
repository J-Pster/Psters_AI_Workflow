# Create Plan

**Objective:** Build an execution-ready implementation plan with clear phases and concrete tasks.

**Inputs:** Feature, bug, or improvement to plan.

**Steps:**
1. Parse the requested outcome and constraints.
2. Research relevant files, patterns, and existing docs.
3. Break work into dependency-ordered phases.
4. Write concrete tasks with exact file targets.
5. Add acceptance criteria and completion checklist.

**Output:** Plan document with phases, tasks, and checklist.
